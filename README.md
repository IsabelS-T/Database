# Database
This is a MySQL database project developed for my database module. It manages the operations of a culinary studio, including clients, instructors, seminars, workshops, and other related entities. It was completed as a group assignment in collaboration with my group members Farzana, Arham, and Jenn. 

---

## How It Was Built
* Database: MySQL
* Language: SQL
* Concepts Used:
    * Database Creation 
    * Data Insertion 
    * Primary & Foreign Keys
    * One-to-Many Relationships
    * Associative Tables
    * Common Table Expressions 
    * Joins
    * Aggregate Functions
    * Subqueries
    * Reports and Data Analysis
